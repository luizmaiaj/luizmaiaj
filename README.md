### What I'm into these days

I build across the stack — from **SwiftUI** apps on Apple platforms to **Python** services and **agentic tooling / MCP servers** that wire large language models into real workflows. I lean toward **local and open-weight models** where it makes sense, and I run a good chunk of my own infrastructure rather than renting it. Lately most of my time goes to LLM-driven automation, native app development, and the plumbing that connects them.

- 🧩 Building **MCP servers** and agent workflows
- 🍎 **SwiftUI** on Apple platforms, plus localization tooling
- 🤖 **Local / open-weight LLM** pipelines
- 🏠 **Self-hosted** infrastructure & home automation
- 🌐 Web / CMS and data dashboards on the side
